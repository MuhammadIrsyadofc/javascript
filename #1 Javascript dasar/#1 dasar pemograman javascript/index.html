dasar pemograman dengan javascript

pengertian pemograman


pemograman dalah kegiatan membuat sebuah program dengan syntax atau aturan untuk menulis kata, syntax pada bahasa pemograman biasanya berbeda beda .

bahasa pemograman

  • bahasa tingkat tinggi (javascript, phyton,java,ruby,VB.NET)
  • bahasa asembly atau bahasa tingkat rendah (contoh : c dan c++)
  • bahasa mesin

penerjemah bahasa

  • compiler
  • interpreter(contoh javascript)
  • intermediate lenguange(IL)/ byte code

compiler

kelebihan dan kekurangan :

kelebihan compiler :

  • siap di jalankan
  • lebih cepat
  • sourcode tidak terlohat

kekurangan compiler :

  • tidak lintas platform
  • tidak flexsibel
  • langkah tambahan

Interpreter

kelebihan interpreter

  • lintas platform
  • fleksibel
  • mudah di uji

kekurangan interpreter

  • butuh interpreter
  • lebih lambat
  • source code mudah di akses

intermediate lenguange/byte code :

menggabungkan kelebihan dari compiler dan interpreter
